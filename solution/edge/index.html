Embedded OPC UA Device Flexing EDGE

Flexing Edge는 OPC UA Server을 통하여 실시간으로 자동 모델링합니다. 쉽고 빠르게 설치할 수 있으며 작동법도 간편합니다. Edge를 사용하여 클러스터링을 통한 분산처리와 다중화 기술을 경험해보세요.

Flexing EDGE Block Diagram

  • OPC UA Framework 기반 Historian, Alarm, Method 탑재
  • 클라우드 연계 기능과 MQTT , PubSub@TSN 기반의 통신
  • Modbus 통신, 사설 비표준 통신 모두 OPC UA 변환
  • 데이터 필터링으로 전송대역 절약
  • 다른 엣지 컴퓨터로 P2P 통신 가능
  • 엣지가 컴퓨터 기능을 하여 네트워크 단절 시 단독 작동
