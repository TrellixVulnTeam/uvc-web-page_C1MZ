OPC Address Space Editor OPC UA Modeler

Flexing Modeler는 세계 최초 OPC UA Address Space 기반으로 CPS를 구현한 모델링 툴(Tool)입니다. Drag & Drop GUI 방식으로 간편한 디바이스 설정을 경험해보실 수 있습니다.

Flexing CPS - OPC UA Modeler diagram

장치/공정 모델링 데이터 무중단 CPS Server 전달
그래픽 환경에서 장치를 국제표준 산업프로토콜인 OPC UA로 모델링
모델러의 장치 모델링과 공정모델링은 즉시 CPS Server에 반영
CPS Server내의 모든 Address Space 데이터 즉시 HTTP 프로토콜로 변환, 웹브라우저 내에서 서비스
OPC NodeSetXML 표준사용 정의
표준 XML 정의로 다른 모델러 XML과 Import / export 가능
OPC 표준에 따른 노드, 메소드, 알람 정의
모델링을 통한 산업용 프로토콜 컨버팅
Modbus, Fanuc, LS XGT 디바이스 OPC UA 프로토콜로 변환
비표준 프로토콜 데이터 파싱 후 OPC UA 프로토콜로 변환 가능
