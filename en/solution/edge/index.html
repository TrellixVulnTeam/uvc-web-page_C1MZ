Embedded OPC UA Device Flexing EDGE

Flexing Edge is automatically modeled in real time through OPC UA Server. It's quick and easy to install and easy to operate. Experience distributed processing and multiplexing technology through clustering using Edge.

Flexing EDGE Block Diagram

  • OPC UA Framework based Historian, Alarm, Method installed
  • Cloud connection function and communication based on MQTT and PubSub@TSN
  • OPC UA conversion for both Modbus communication and private non-standard communication
  • Data filtering saves transmission bandwidth
  • P2P communication with other edge computers
  • The edge functions as a computer and operates independently when the network is disconnected
